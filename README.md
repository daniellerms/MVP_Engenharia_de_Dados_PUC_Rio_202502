# Análise de Microdados Enem 2024

Esse projeto tem como objetivo analisar os microdados do Enem 2024 e responder algumas sobre dados demograficos dos participantes e a influência de fatores socieconomico no desempenho nas provas.

As perguntas a serem respondidas são: 

1) Quantidade de inscritos.
2) Quantidade de inscritos por estado.
3) Quantidade de inscritos na rede pública e privada.
4) Quantidade de faltantes.
5) Qual perfil dos faltantes.
6) Notas por localidade.
7) Notas por perfil socioeconomicos, como nível de eduação dos pais


Os dados utilizados estão disponíveis nos site gov.com, no link: https://www.gov.br/inep/pt-br/acesso-a-informacao/dados-abertos/microdados/enem
